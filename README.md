# Blood-Bank-Management-System
Blood Bank Management System (BBMS) isÂ a web based system that can assists the information of blood bag during its handling in the blood bank. With this system, the user of this system can key in the result of blood test that has been conducted to each of the blood bag received by the blood bank.
